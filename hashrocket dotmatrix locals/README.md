# My dotfiles

## These dotfiles are my personal preferences on top of Hashrocket's dotmatrix

Symlink the dotfiles to the home directory with the extension .local and dotmatrix's dotfiles will expand them
  
For zsh, I use oh-my-zsh for theme and plugin support. To install,
  
	1. Grab the oh-my-zsh framework
		git clone git://github.com/robbyrussell/oh-my-zsh.git ~/.oh-my-zsh
	2: Symlink this zshrc to your home directory, appended with .local
